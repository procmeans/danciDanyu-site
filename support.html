常见问题

AI 功能是免费的吗？

AI 学习助手（单词智能拆解、学习诊断报告、自定义词集生成、AI 学习对话）需要 9.9 元 / 月订阅。如果你已经有 DeepSeek API Key，可以在「我的 → 设置 → AI 智能分析 → 使用自己的 API Key」处粘贴，免费使用全部 AI 功能。

其他所有功能（词典、词根、12 种练习、SM-2 复习、学习统计、收藏）永久免费、无广告。

怎么取消订阅？

iOS 系统「设置 → Apple ID → 订阅」里取消，取消后当月仍可使用至期满，到期后自动停止扣费。

恢复购买在哪里？

「我的 → 设置 → AI 学习助手 → 订阅 AI 学习助手」页面底部有「恢复购买」按钮。换设备或重装 App 后用同一 Apple ID 点一下即可恢复订阅状态。

学习数据会同步到其他设备吗？

目前不会。所有学习记录、收藏、自定义词集都存在本机。换设备需要重新开始学习记录。iCloud 同步功能正在规划中。

词条 / 例句里发现错误怎么办？

非常欢迎反馈。把错的词、当前 App 显示的内容、你认为正确的内容用邮件或微信发给我，会在下个版本修正。

清空学习数据后还能恢复吗？

不能。「我的 → 设置 → 数据 → 清空学习数据」会永久删除本机的所有学习进度、复习记录、收藏。请谨慎操作。

AI 调用会泄露我的隐私吗？

AI 调用只把当前查询的英文单词或简短的学习摘要发送给 DeepSeek，不发送账号、设备 ID、邮箱、手机号等任何身份信息。在「设置 → 启用 AI 功能」处可以一键关闭所有联网行为。详见隐私政策。

支持哪些 iOS 版本？

iOS 17 及以上。iPhone、iPad 通用。
