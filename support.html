常见问题

App 是免费的吗?

是的,核心功能完全免费。AI 智能拆解、AI 学习诊断、AI 学习助手等基于大模型的功能采用「免费积分 + 自带 Key」的方式:新用户赠送一次性 100 积分;用完后可购买积分,或在「我的 → 设置 → 使用自己的 API Key」里粘贴自己的 DeepSeek Key 免积分使用。

关闭 AI 后,App 还会联网吗?

不会。在「我的 → 设置 → AI 智能分析」关闭 AI 功能后,App 完全离线运行 — 词典数据、SM-2 复习算法、所有游戏、统计图表都在本机完成,不向任何服务器发送数据。

怎么备份学习数据?

当前 v1 不支持显式导出。所有数据保存在 iOS 沙盒里,会随 iCloud 设备备份一起被备份(如果你开启了"App 数据" iCloud 备份)。卸载 App 会清除全部本地数据,请谨慎。下个版本计划加入 JSON / CSV 导出。

我可以删除自己的数据吗?

可以,有三种方式:

  • 「我的 → 设置 → 数据 → 清空学习记录」:清掉 SM-2 复习状态 + 打卡计数。
  • 「我的 → 设置 → 数据 → 清空收藏与历史」:清掉收藏单词 + 搜索历史。
  • 在系统设置中卸载本 App:清掉本机所有数据。

每日提醒为什么没有响?

请确认:

  • 「我的 → 设置 → 每日复习提醒」是开启状态。
  • iOS 系统「设置 → 通知 → 单词单语」中"允许通知"为开。
  • 勿扰模式 / 专注模式没有屏蔽通知。

词义不准确 / 词典里词没找到?

词典数据来自开源的 ECDICT、Webster's 1913、Wiktionary、WordNet 等,我们尽量校对,但难免有疏漏。如果你发现具体错误,直接邮件 / 微信告诉我们具体单词和你认为正确的释义,我们会修正并在下个版本更新。

iCloud 同步什么时候有?

v1 没有跨设备同步。该功能在 v1.1 之后的路线图上,需要先做好数据迁移和冲突解决方案,我们不希望草率地推出会丢数据的同步。

找不到某个游戏模式?

有些游戏模式只在特定主题词集下可用 — 比如「动物分类」只在「动物与野生生物」主题下出现,「切水果」只在「水果与坚果」下出现,「温室」只在植物相关的几个主题下出现。这是设计上的取舍,让每种玩法都能跟单词内容匹配。

退款 / 取消订阅?

本 App 不订阅。如果你购买了积分包,但希望退款,请通过 App Store 的标准退款流程申请(报告问题)。Apple 处理所有退款。
