单词单语 · 隐私政策

生效日期:2026-05-21(v1.0) · 最近修订:2026-06-01

「单词单语」(下称"我们"或"本应用")非常重视您的隐私。本政策说明在您使用本应用 iOS 端时,我们收集、使用、存储哪些信息,以及您对这些信息拥有的权利。如果您不同意本政策的任何条款,请勿安装或使用本应用。

一、我们不收集什么

本应用的核心练习功能完全离线。我们:

  • 不收集您的个人身份信息(姓名、手机号、邮箱、身份证号等);
  • 不收集您的设备识别符(IDFA、IDFV、UUID 等);
  • 不收集您的位置信息、通讯录、相册或日历;
  • 不向任何第三方服务器发送您的学习记录、收藏单词或搜索历史;
  • 不集成任何第三方分析、广告或追踪 SDK;
  • 不要求您注册账号或登录。

二、我们在本地存储什么

应用为了实现核心功能,会在您的设备本地(沙盒目录)保存以下信息:

学习进度(已完成的词集和模式)UserDefaults显示完成徽章、最近练习入口

以上所有数据仅存储在您本机的应用沙盒中,卸载应用即随系统机制清除。我们无法在服务端访问、备份或同步这些数据。

您可以随时在"我的 → 设置 → 数据"中:

  • 清空学习记录(SM-2 状态 + 活动计数);
  • 清空收藏与历史(收藏单词 + 搜索历史)。

三、AI 功能与数据传输

AI 智能拆解、AI 学习诊断、AI 学习助手是可选功能,默认开启但可以在 "我的 → 设置 → AI 智能分析"中关闭。开启时:

  • 内置积分路径:应用通过 HTTPS 调用我们自己的代理服务器,代理再调用 DeepSeek 模型。 发送内容包括:当前查询的单词与释义、当前对话(在 AI 助手内输入的文字)、为生成学习诊断而聚合的统计摘要(累计复习次数、掌握/学习中数量、易忘词列表 — 不含设备身份信息或账户信息)。 不发送:您的姓名、邮箱、设备识别符、位置、其他单词集的完整数据。
  • BYOK 路径(自带 API Key):在"我的 → 设置 → 使用自己的 API Key"填入自己的 DeepSeek Key 后,应用直接调用 DeepSeek API,绕过我们的代理。此模式下您的内容只在您和 DeepSeek 之间传递,我们的服务器不参与。Key 仅保存在本机 Keychain。
  • AI 调用都不会在我们服务端长期保留对话内容;代理仅做请求转发与短期速率限制必需的日志。
  • 如完全不希望使用 AI,关闭"AI 智能分析"开关后,应用不会向我们或 DeepSeek 发送任何数据。

四、通知权限

如果您在"我的 → 设置 → 每日复习提醒"中启用每日提醒,应用会调用 iOS 系统通知 API (UNUserNotificationCenter)在您指定的本地时间投递一条本地通知。该通知仅在您的设备本机生成,不经过任何远程推送服务,亦不携带广告或营销内容。您可以随时关闭。

五、第三方词典数据归属

应用使用以下开源词典数据,已在"我的 → 致谢"中标注:

  • ECDICT(MIT)— 中文释义主要来源;
  • Webster's 1913(公有领域)— 英文释义主要来源;
  • English Wiktionary(CC BY-SA 4.0)— 英文释义备用来源;
  • Princeton WordNet 3.0(WordNet License)— 同义词组归类数据来源。

我们尊重并履行上述协议规定的归属要求。词典数据随应用一同分发,运行时不联网下载。

六、儿童隐私

本应用面向 4 岁及以上所有年龄段的英语学习者。由于我们不收集任何个人信息,亦不要求注册账号,应用本身不存在儿童信息泄露的风险。家长可通过 iOS "屏幕使用时间"管理孩子的使用时长。

七、您的权利

由于本应用不在服务端存储您的任何数据,您无需向我们提交任何请求即可:

  • 查看您的数据:在应用内即可看到所有;
  • 删除您的数据:使用"我的 → 设置 → 数据"中的清空按钮,或在系统设置中卸载本应用;
  • 导出您的数据:当前版本暂不提供导出功能,未来版本计划支持 JSON / CSV 导出。

八、政策变更

如本政策有重要修改,我们会在应用内"我的 → 关于"中显著标示新版本生效日期。继续使用应用即视为您接受变更后的条款。
